## 📋 WHAT YOU NEED TO DO

### Minimum Setup (Basic App - 7 minutes):

1. **Create Supabase project** (5 min)
   - Go to supabase.com
   - Create project
   - Run `backend/supabase/schema.sql`

2. **Create frontend `.env`** (1 min)
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   VITE_API_URL=http://localhost:3001/api
   ```

3. **Start app** (1 min)
   ```bash
   npm start
   ```

**✅ Done! Basic app works!**

---

### Full Setup (All Features - 30 minutes):
